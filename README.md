# TB Multimodal Diagnosis System

This repository implements a multimodal tuberculosis diagnosis framework
using chest X-ray images and clinical tabular data.

## Components
- Image model: EfficientNetV2
- Tabular model: SAINT-like Transformer
- Fusion model: Feature-level fusion
- Explainability: Grad-CAM++

## Note
Trained weights, datasets, and final hyperparameters are excluded and
will be released after publication.
